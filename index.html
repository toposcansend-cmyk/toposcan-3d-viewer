🗼 TopoScan · Visualizador 3D

Visualizador interativo da Torre Radar Meteorológico Banda C — projeto da estação SIMEPAR (Sistema Meteorológico do Paraná). Modelagem 3D paramétrica feita pela TopoScan a partir do PDF arquitetônico.

⚙️ Stack técnico

  • Modelagem: Python + trimesh (paramétrico, regenerável)
  • Renderização programática: pyvista / VTK
  • Visualizador web: Google model-viewer 3.5 (Three.js + WebXR)
  • Hosting: GitHub Pages (CI/CD automático)
  • Pipeline point cloud: laspy + Potree (prep para 5GB+)

📊 Dados do modelo

  • 401 geometrias individuais
  • 15 grupos de componentes
  • Bounding box: 26,15 × 14,15 × 30,67 m
  • Origem: centro da base da torre, eixos X=leste, Y=norte, Z=up
  • Unidades: metros
